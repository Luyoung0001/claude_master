# 第五章：MCP Server 集成

MCP（Model Context Protocol）是一个开放协议，让 Claude Code 能够与外部工具和数据源集成，大大扩展其能力。

## 为什么使用 MCP？

| 特性 | 内置工具 | MCP Server |
|------|---------|------------|
| 文件访问 | 基本读写 | 高级过滤、监控 |
| 数据库 | ❌ | ✅ 直接查询 |
| 浏览器 | 基本获取 | 完整自动化 |
| API 集成 | 手动调用 | 封装好的工具 |
| 自定义能力 | ❌ | ✅ 完全可扩展 |
