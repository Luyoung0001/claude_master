# 第一章：基础入门

本章将带你从零开始掌握 Claude Code CLI 的基础使用方法。

## 本章内容

- [1.1 安装与配置](./01-installation.md)
- [1.2 基本命令](./02-basic-commands.md)
- [1.3 会话管理](./03-session-management.md)

## 学习目标

完成本章后，你将能够：

1. 正确安装和配置 Claude Code CLI
2. 理解并使用所有基本命令
3. 有效管理多个会话
4. 根据需求选择合适的模型

## 前置要求

- Node.js 18+ 或更高版本
- 有效的 Anthropic API Key（或自定义端点）
- 基本的命令行操作知识

## 预计学习时间

约 2-3 小时
