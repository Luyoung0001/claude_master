# 第四章：Hooks 钩子系统

Hooks（钩子）是 Claude Code 的强大自动化功能，让你能够在工具执行前后运行自定义脚本，实现工作流自动化。

- [4.2 实用 Hook 示例](./02-hook-examples.md)

## 学习目标

完成本章后，你将能够：

1. 理解不同类型的 Hook 及其触发时机
2. 配置 Hook 匹配器精确控制触发条件
3. 编写实用的自动化脚本
4. 构建复杂的工作流自动化

## Hook 概览

| Hook 类型 | 触发时机 | 用途 |
|-----------|---------|------|
| `PreToolUse` | 工具执行前 | 验证、准备、拦截 |
| `PostToolUse` | 工具执行后 | 格式化、测试、通知 |
| `Notification` | 通知事件时 | 外部通知、日志 |
| `Stop` | 任务停止时 | 清理、报告 |

## Hook 工作流程

```
用户输入 → Claude 处理 → 调用工具
                           ↓
                    ┌──────────────┐
                    │ PreToolUse   │ ← 可以拦截/修改
                    └──────────────┘
                           ↓
                    ┌──────────────┐
                    │  工具执行     │
                    └──────────────┘
                           ↓
                    ┌──────────────┐
                    │ PostToolUse  │ ← 可以添加处理
                    └──────────────┘
                           ↓
                      返回结果
```
