## SDK 概览

```
┌──────────────────────────────────────────────────┐
│                 Your Application                  │

│  └────────────┘ └────────────┘ └────────────┘   │
├──────────────────────────────────────────────────┤

```

## 为什么使用 Agent SDK？

| 特性 | 直接 API 调用 | Agent SDK |
|------|--------------|-----------|
| 工具循环 | 手动实现 | 自动处理 |
| 会话管理 | 手动维护 | 内置支持 |
| 错误处理 | 自行处理 | 统一处理 |
| 类型安全 | 无 | 完整类型 |
