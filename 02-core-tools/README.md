# 第二章：核心工具使用

Claude Code 内置了一套强大的工具集，让 AI 能够直接操作文件、执行命令、搜索网络等。本章将深入介绍每个工具的使用方法。

## 本章内容

- [2.1 文件操作工具](./01-file-tools.md)
- [2.2 代码执行工具](./02-bash-tools.md)
- [2.3 Web 工具](./03-web-tools.md)
- [2.4 任务管理工具](./04-task-tools.md)
- [2.5 交互工具](./05-interaction-tools.md)

## 学习目标

完成本章后，你将能够：

1. 熟练使用 Read/Write/Edit 进行文件操作
2. 使用 Glob/Grep 进行高效搜索
3. 通过 Bash 工具执行系统命令
4. 使用 Web 工具获取网络信息
5. 使用 Task 工具启动子 Agent 处理复杂任务

## 工具概览

| 类别 | 工具 | 功能 |
|------|------|------|
| 文件操作 | Read | 读取文件内容 |
| | Write | 创建/覆盖文件 |
| | Edit | 精确编辑文件 |
| | Glob | 文件模式匹配 |
| | Grep | 内容搜索 |
| 代码执行 | Bash | 执行 shell 命令 |
| Web | WebSearch | 网络搜索 |
| | WebFetch | 获取网页内容 |
| 任务管理 | Task | 启动子 Agent |
| | TodoWrite | 任务列表管理 |
| 交互 | AskUserQuestion | 向用户提问 |

## 预计学习时间

约 4-5 小时
