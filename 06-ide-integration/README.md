# 第六章：IDE 集成

Claude Code 可以与多种 IDE 和编辑器集成，提供无缝的开发体验。

## 本章内容

- [6.1 VS Code 集成](./01-vscode.md)
- [6.2 JetBrains IDE 集成](./02-jetbrains.md)
- [6.3 其他编辑器](./03-other-editors.md)

## 学习目标

完成本章后，你将能够：

1. 在 VS Code 中使用 Claude Code 扩展
2. 在 JetBrains IDE 中使用 Claude 插件
3. 在 Vim/Neovim 和其他编辑器中集成 Claude Code
4. 配置快捷键和自定义工作流

## IDE 支持概览

| IDE | 集成方式 | 特性 |
|-----|---------|------|
| VS Code | 官方扩展 | 内联聊天、侧边栏、终端集成 |
| JetBrains | 官方插件 | 代码建议、聊天窗口 |
| Vim/Neovim | 终端 + 插件 | 命令行集成 |
| Zed | 内置支持 | 原生 AI 功能 |
| Emacs | 终端 + 包 | Shell 集成 |

## 预计学习时间

约 2-3 小时
